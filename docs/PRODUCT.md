# PRODUCT

## 제품명

가칭: JLPT 스타일 일본어 단어 학습 앱

## 제품 한 줄 설명

짧은 세션으로 일본어 단어를 외우고, 틀린 문제를 다시 복습하게 만드는 미니멀한 일본어 학습 앱

## 목표

- 빠른 MVP 출시
- 광고 기반 수익화
- 저작권 문제 없는 오픈 데이터 활용
- 로그인 없이 바로 사용 가능
- 서버 의존도를 최소화하고 로컬 중심 구조 채택
- 짧고 반복적인 학습 경험 제공

## 타겟 사용자

- JLPT N5~N3 중심의 일본어 학습자
- 출퇴근, 이동 중, 쉬는 시간에 짧게 공부하는 사용자
- 복잡한 가입 절차 없이 바로 학습하고 싶은 사용자
- 기존 공부앱의 딱딱한 UX보다 가볍고 예쁜 UI를 선호하는 사용자

## MVP 범위

- N5 단어 중심
- 의미 맞추기 퀴즈
- 읽기 맞추기 퀴즈
- 오답 저장
- 다음 세션 우선 복습
- OS 내장 TTS 재생
- 로컬 저장
- 광고 삽입

## 제외 범위

- 로그인
- 계정 연동
- 클라우드 동기화
- 친구 기능
- 랭킹
- 채팅
- 문법 강의형 콘텐츠
- 서버 기반 개인화 추천

## 핵심 가치

- 바로 시작 가능
- 짧게 공부 가능
- 눈이 편안함
- 귀엽지만 유치하지 않음
- 복습이 자동으로 이어짐

## 성공 기준

- 사용자가 1회 세션을 1~2분 안에 끝낼 수 있음
- 첫 실행 후 회원가입 없이 바로 퀴즈 시작 가능
- 틀린 문제가 다음 세션에서 자연스럽게 재노출됨
- TTS가 기기 기본 기능으로 동작함
- 대부분의 데이터와 학습 상태가 로컬에 저장됨

## 배포 정책

- 테스트 앱 배포는 `0.xx` 버전으로 진행
- 테스트 종료 후 정식 배포는 `1.0.0`으로 진행
