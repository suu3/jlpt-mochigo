# CHARACTER SPEC

## 캐릭터 종류

토끼

## 역할

학습을 방해하지 않는 작은 동반자 캐릭터

## 캐릭터 목표

- 사용자에게 정서적 온기를 제공
- 부담을 줄이고 친근함을 더함
- 공부를 강요하지 않고 옆에서 응원하는 느낌

## 디자인 방향

- 작고 둥근 형태
- 부드러운 귀 형태
- 단순한 얼굴
- 작은 눈, 작은 입
- 파스텔 톤 사용
- 세미플랫 또는 플랫 일러스트 스타일

## 감정 표현

- 기본: 차분한 표정
- 정답: 작은 미소
- 오답: 살짝 위로하는 표정
- 결과: 조용한 기쁨

## 금지

- 화면 중앙 점유
- 과한 애니메이션
- 너무 유아용 스타일
- 시선 강탈형 연출

## 배치 위치

- 홈 상단 또는 구석
- 퀴즈 화면 상단 구석
- 결과 화면 헤더 주변
