# FEATURE SPEC

## 1. 홈 화면

### 포함 요소

- 앱 타이틀
- JLPT 레벨 선택 버튼
- Daily Goal 진행률
- 연속 학습 일수
- 시작 버튼
- 오답 복습 진입 버튼

### 동작

- 최근 선택 레벨 기억
- 첫 MVP에서는 N5를 기본 강조

---

## 2. 퀴즈 화면

### 공통 요소

- 현재 문제 번호 표시 (예: 3 / 5)
- 퀴즈 타입 라벨
- 문제 텍스트
- 발음 버튼
- 4개의 선택지 버튼

### 문제 타입 A: 의미 맞추기

- 문제: 일본어 단어 제시
- 선택지: 한국어 의미 4개

### 문제 타입 B: 읽기 맞추기

- 문제: 한자 또는 단어 제시
- 선택지: kana 읽기 4개

### 정답 처리

- 탭 즉시 결과 표시
- 짧은 시간 후 자동 다음 이동

### TTS

- 사용자가 발음 버튼을 눌렀을 때만 실행
- 자동 재생 없음
- OS 내장 일본어 TTS 사용
- 실패 시 앱이 멈추지 않고 조용히 무시하거나 안내 표시

---

## 3. 결과 화면

### 포함 요소

- 맞은 개수
- 짧은 칭찬 메시지
- 틀린 문제 미리보기
- 다시하기 버튼
- 홈으로 버튼

### 광고

- 세션 종료 후 전면 광고 노출 가능

---

## 4. 오답 복습 화면

### 포함 요소

- 틀린 단어 목록
- 복습 시작 버튼
- 간단한 필터 또는 정렬 기준

### 동작

- 로컬 저장 기반
- 다음 세션 우선 출제와 별도로 사용자가 수동 진입 가능

---

## 5. 로컬 저장

### 저장 대상

- 최근 선택 레벨
- Daily Goal 진행률
- 연속 학습 일수
- 오답 기록
- 간단한 학습 이력
- 설정값(소리, TTS 허용 여부 등)

### 저장 원칙

- 로그인 없이 동작
- 가능하면 서버 저장 없이 로컬 우선
- 앱 삭제 시 학습 이력은 삭제될 수 있음

---

## 6. 광고

### MVP 광고 포인트

- 세션 결과 화면 이후 전면 광고
- 힌트 사용 시 보상형 광고

### 광고 원칙

- 학습 중간 강제 방해 최소화
- 세션 종료 지점에 자연스럽게 삽입

---

## 7. 토끼 캐릭터

### 역할

- 홈 화면에서 환영
- 정답 시 작게 기뻐함
- 오답 시 조용히 위로함
- 결과 화면에서 칭찬

### 원칙

- UI를 방해하지 않음
- 화면 구석 또는 헤더 위주 노출
- 과한 표정, 과한 움직임 금지

## 8.Language Support

### 지원 언어

- 한국어
- 영어

### 적용 범위

- UI 텍스트
- 버튼, 라벨, 메시지
- 결과 화면 메시지

### 미포함 (MVP)

- 다국어 음성
- 사용자 선택 언어 저장 기능 (선택적으로 추가 가능)

### 기본 정책

- 기기 언어를 자동 감지
- 지원 언어가 아닐 경우 영어 fallback
