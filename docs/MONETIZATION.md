# MONETIZATION

## 목표

광고 기반으로 수익화하되 학습 흐름을 해치지 않는다.

## 광고 종류

- 하단 배너 광고
- 전면 광고
- 보상형 광고

## 광고 위치

### 하단 배너 광고

- 메인 화면 공통 하단 영역
- 학습 진행을 막지 않도록 하단 네비게이션 위에 고정된 슬롯으로 최소 노출

### 전면 광고

- 세션 종료 후 결과 화면 진입 전 또는 후

### 보상형 광고

- 힌트 기능 사용 시
- 이어서 복습 강화 기능 제공 시

## 광고 원칙

- 배너는 한 화면에 1개만 유지
- 문제 풀이 중간 강제 광고 금지
- 세션 단위 흐름을 깨지 않게 배치
- 너무 잦은 노출 금지

## 향후 확장

- 광고 제거 유료 옵션
- 프리미엄 테마
- 추가 복습 기능 유료화
