# UX UI SPEC

## 전체 톤

- 눈이 편안한 UI
- 세련되고 미니멀한 구성
- 은은하게 귀여움
- 공부앱 같기보다 가벼운 감성 앱 느낌
- 차분하고 부드러운 인상

## 시각 원칙

- 과한 대비 피하기
- 순백/순검정보다 오프화이트와 다크그레이 사용
- 여백 충분히 확보
- 버튼은 둥글고 터치하기 쉽게
- 화면 요소 수를 최소화

## 색감 방향

- 저채도 민트, 티얼, 파스텔 블루 계열 포인트
- 배경은 웜 라이트 그레이 또는 오프화이트
- 피드백 색상도 눈에 자극적이지 않게 조절

## 타이포 원칙

- 가독성 우선
- 일본어와 한국어가 같이 보여도 깨지지 않는 폰트 선택
- 제목과 단어는 크게
- 설명은 짧고 명확하게

## 인터랙션 원칙

- 한 손 사용 가능
- 다음 행동이 항상 명확해야 함
- 부드럽지만 빠른 화면 전환
- 애니메이션은 최소한으로 사용

## 감정 목표

- 편안하다
- 예쁘다
- 부담 없다
- 조금 더 풀고 싶다
